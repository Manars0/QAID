# QAID

AI-powered financial risk analysis and IFRS compliance platform.

## Overview

QAID analyzes accounting data to detect financial anomalies, assess compliance with IFRS standards, and generate AI-powered audit insights through an interactive dashboard.


## Features

- Financial anomaly detection
- IFRS compliance assessment
- Fraud indicators
- Risk scoring
- AI-generated audit insights
- Interactive dashboard
- PDF report generation
- Arabic & English support
- Light / Dark mode

## Dataset

The application uses a **synthetic financial dataset** generated specifically for this project.

Dataset generation repository:

> [Dataset Repository][https://github.com/Manars0/erp-fraud-detection-system]

The generated dataset simulates realistic accounting records while preserving privacy and avoiding the use of real financial information.
